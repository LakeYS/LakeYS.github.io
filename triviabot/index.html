About

TriviaBot brings multiplayer trivia games to your Discord server! With over 2,500 questions and 24 categories, TriviaBot offers endless trivia fun.

Categories include Film, Television, Anime & Manga, Sports, Science & Nature, and more.

How to Use

To use the bot, simply add it to your server and type trivia help in a text channel.

Manual installation and hosting instructions coming soon.

Development and Support

Note: Development of this project will be slow through the month of October as I am be working hard on another project. Normal development will resume soon.

TriviaBot is powered by discord.js and the Open Trivia Database (OpenTDB). This project is still a work-in-progress, so there are plenty of new features on the way!

For support, questions, and comments, you can join the Discord server or contact me directly at contact@LakeYS.net. You can also submit an issue on GitHub for bugs and support.
